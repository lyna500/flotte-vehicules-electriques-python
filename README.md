
# Projet Python – Gestion d'une flotte de véhicules électriques

## Description
Application Python pour gérer une flotte de véhicules électriques pour une entreprise de location courte durée.  
Fonctionnalités principales :
- Gestion des véhicules (ajout, modification, suppression, recherche)
- Gestion des utilisateurs (Clients et Gestionnaires)
- Gestion des locations (démarrer, terminer)
- Gestion des maintenances
- Sauvegarde et chargement des données (JSON et CSV)
- Statistiques et graphiques avec pandas et matplotlib
- Interface en ligne de commande (CLI)
